プライバシーポリシー

このプライバシーポリシーは、Chrome拡張機能「AWS Account Color Overlay」がユーザーデータをどのように収集、使用、保存するかを説明するものです。

1. 収集する情報

本拡張機能は、以下の情報を収集します。

  • 拡張機能内で設定されるデータ: ユーザーがオプションページで設定するAWSアカウントID、アカウント名、色、オーバーライド設定などのアカウントマッピング情報、およびオーバーレイの表示設定（位置、色、フォントなど）、境界線の幅、ダークモードの好み。これらのデータはユーザーのデバイス上でローカルに生成・設定されます。
  • 自動的に収集されるデータ:
    • ウェブサイトのコンテンツ: 現在のAWSコンソールページから、AWSアカウントID、アカウント名、リージョン、および動的な色情報といった特定のテキストコンテンツを読み取ります。これは拡張機能のコア機能（オーバーレイ表示や境界線の色変更）に不可欠です。画像、音声、動画、ハイパーリンクなどの他のウェブサイトコンテンツは収集しません。
    これらのデータもユーザーのデバイス上でローカルに処理されます。

2. 収集の目的

収集された情報は、以下の目的のためにのみ使用されます。

  • AWSマネジメントコンソール上で、現在のアカウントID、アカウント名、リージョン、および関連する色情報をオーバーレイとして表示するため。
  • ユーザーが設定したカスタマイズ（アカウントマッピング、オーバーレイのスタイル、境界線の色など）を適用し、拡張機能の機能を提供するため。

3. データの保存

収集されたデータは、chrome.storage.sync または chrome.storage.local を使用して、ユーザーのブラウザのローカルストレージにのみ保存されます。これらのデータは、ユーザーのChromeプロファイルと同期される場合がありますが、Googleのインフラストラクチャ内で安全に管理されます。

4. データの共有と第三者への提供

本拡張機能は、ユーザーのデバイスから外部にデータを送信することは一切ありません。 したがって、収集されたユーザーデータは、いかなる第三者とも共有、販売、転送されることはありません。

5. 分析、追跡、リモートロギング

本拡張機能は、ユーザーの行動を追跡するための分析ツール、追跡ツール、またはリモートロギングサービスを一切使用していません。

6. ユーザーのデータの管理

ユーザーは、拡張機能のオプションページを通じて、保存されているアカウントマッピングや設定をいつでも変更または削除できます。拡張機能をアンインストールすることで、本拡張機能が保存したすべてのデータは削除されます。

7. プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて変更されることがあります。変更があった場合は、拡張機能の更新を通じて通知されます。

8. お問い合わせ

本プライバシーポリシーに関するご質問やご不明な点がございましたら、[開発者の連絡先情報] までお問い合わせください。

Privacy Policy

This Privacy Policy explains how the Chrome extension "AWS Account Color Overlay" collects, uses, and stores user data.

1. Information Collected

This extension collects the following information:

  • Data configured within the extension: Account mapping information such as AWS Account ID, account name, color, and override settings, as well as overlay display settings (position, color, font, etc.), border width, and dark mode preference, which are set by the user on the options page. This data is generated and configured locally on the user's device.
  • Automatically collected data:
    • Website Content: Reads specific text content such as AWS Account ID, account name, region, and dynamic color information from the current AWS console page. This is essential for the core functionality of the extension (overlay display and dynamic border coloring). It does not collect other website content such as images, audio, video, or hyperlinks.
    This data is also processed locally on the user's device.

2. Purpose of Collection

The collected information is used solely for the following purposes:

  • To display the current account ID, account name, region, and associated color information as an overlay on the AWS Management Console.
  • To apply user-configured customizations (account mappings, overlay styles, border colors, etc.) and provide the functionality of the extension.

3. Data Storage

Collected data is stored only in the user's browser's local storage using chrome.storage.sync or chrome.storage.local. This data may be synchronized with the user's Chrome profile but is securely managed within Google's infrastructure.

4. Data Sharing and Provision to Third Parties

This extension does not transmit any data outside the user's device. Therefore, collected user data is never shared, sold, or transferred to any third party.

5. Analytics, Tracking, and Remote Logging

This extension does not use any analytics tools, tracking tools, or remote logging services to track user behavior.

6. Management of User Data

Users can change or delete their saved account mappings and settings at any time through the extension's options page. Uninstalling the extension will delete all data saved by this extension.

7. Changes to this Privacy Policy

This Privacy Policy may be updated as necessary. Any changes will be notified through extension updates.

8. Contact Information

If you have any questions or concerns regarding this Privacy Policy, please contact [Developer Contact Information].
